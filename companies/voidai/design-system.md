## Image Guidelines

- All social images use dark background with brand accent overlay
- Data cards use Space Grotesk for numbers, Inter for labels
- Minimal emoji usage in graphics (matches main account voice)
- Clean, infrastructure-focused visual language. No cartoons or overly playful graphics on main account content

## X (Twitter) Visual Identity Standards

Consistency across all visual touchpoints on X builds recognition. When someone sees a VoidAI post, they should recognize it before reading the text.

### Profile Consistency (All VoidAI-Controlled Accounts)

| Element | @v0idai Standard | Satellite Accounts |
|---------|-----------------|-------------------|
| Profile picture | VoidAI logo mark on dark background. Do NOT change frequently. Consistency builds recognition | Unique per persona. Must NOT use VoidAI branding (organic appearance). Update no more than 1x/quarter |
| Banner image | Current product hero or key metric visual. Update only for major launches (lending, new chain). Always include one-liner tagline | Persona-appropriate. Must include @v0idai mention somewhere visible |
| Bio | One-liner tagline + key product links. Keep current. Update within 24 hours of any major launch | Per-account persona. Must include @v0idai affiliation disclosure |
| Pinned tweet | Always the most important current CTA: latest major launch, or key product link. Update within 24 hours of major announcements | Persona-specific. Must include VoidAI community disclosure |

### Post Image Standards

- **Data cards**: Always use the VoidAI template. Dark background, Space Grotesk for headline numbers, Inter for body text, brand accent for key metrics. Include VoidAI logo watermark (small, bottom-right corner)
- **Announcement graphics**: Consistent layout template. Product name prominent. No stock photos. Use abstract/infrastructure visual language
- **Thread headers**: If a thread includes an image, it goes on the first tweet only. Do not add images to every tweet in a thread (looks spammy)
- **Screenshots**: When sharing product screenshots (bridge UI, staking interface), add a subtle branded border or overlay. Raw screenshots look unprofessional

### Visual DON'Ts for X

- No laser-eye profile pictures
- No price chart screenshots or candlestick images
- No generic stock crypto imagery (gold coins, rocket ships, diamond hands)
- No memes on the main @v0idai account (route to meme satellite)
- No walls of emojis in any visual or text content
- No "announcement" graphics that are just text on a colored background. If it's just text, post it as text
