# Design System: VoidAI

## Visual Identity

| Element | Specification |
|---------|--------------|
| Background | Dark (#0A0A0F or similar) |
| Typography (Headlines) | Space Grotesk |
| Typography (Body) | Inter |
| Primary accent color | VoidAI primary accent |
| Success color | Green |
| Warning color | Amber |

## Partner Badges

- "Built on Chainlink CCIP" where relevant
- "Powered by Bittensor" where relevant

## Image Guidelines

- All social images use dark background with brand accent overlay
- Data cards use Space Grotesk for numbers, Inter for labels
- Minimal emoji usage in graphics (matches main account voice)
- Clean, infrastructure-focused visual language. No cartoons or overly playful graphics on main account content

## Social Media Formats

| Platform | Format | Dimensions |
|----------|--------|-----------|
| X (Twitter) | Single image | 1200x675 |
| X (Twitter) | Thread header | 1200x675 |
| LinkedIn | Post image | 1200x627 |
| Blog (voidai.com) | Hero image | 1200x630 |
