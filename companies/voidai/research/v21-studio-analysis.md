### Design Principles to Adopt

- **Clarity over decoration**: Every UI element should serve a purpose
- **Data density done right**: Show metrics without overwhelming — progressive disclosure
- **Consistent visual system**: Unified color, type, spacing, and component library across all touchpoints
- **Web3-native affordances**: Chain indicators, transaction status patterns, wallet connection flows that users expect
