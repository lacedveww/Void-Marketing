## How to Use This File

The Intelligence Sweep reads this file at 8AM and 8PM ET. For marketing accounts:
1. Check for new posts with marketing tactics or strategies
2. Analyze: content format, hook style, engagement patterns, posting cadence
3. Note specific tactics that could be adapted for VoidAI content strategy
4. Flag high-performing formats or viral mechanics
5. Compare engagement rates across similar account sizes

### What to Capture

For each notable post:
- Format used (thread, single tweet, image, video, poll)
- Hook type (question, stat, contrarian take, story)
- Engagement numbers relative to follower count
- Specific tactic or framework shared
- Adaptability score (1-5) for VoidAI use case

### Adding New Accounts

When adding, include:
- Verified X handle
- Brief description of what they do
- What specifically to learn from them
- Priority level
- Date added
